Eventful

Technologies Used:

HTML5, CSS3, Javascript, Node.js, Express Handlebars, Express, MySQL, Sequelize, Passport

Description:

This app was built in order to give users a place to keep their events organized and add information about the event, the items needed, and guests invited. The user has the chance to sign up in order to be able to come back at a later time and see their saved information. The user can create a new event or edit old ones. In each event, the user can add items needed, guests that will be attending the event, and edit event details. They can delete guests or mark items as checked off to keep track of what completed and what is still needed. This website is mobile responsive to be able to use it on the go!

Loteria Click Game

Technologies Used:

HTML5, CSS3, Javascript, Node.js, React

Description:

This is a memory game created with React. The application renders different images to the screen and which each listen for click events. Every time an image is clicked, if the image has not been clicked before, the images rendered to the page get shuffled and the user's score is incremented. If an image is clicked more than once, the user's score resets to 0 and the game restarts. This React application was created using the Create React App. The application's UI is broken up into components, manage component state, and respond to user events.
